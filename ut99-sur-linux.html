Linux tournament Unreal

Epyc a mis à dispotion gratuitement Unreal & Unreal Tournament pour le projet OldUntreal. Sur le site on y trouve les patchs qui modernisent ses deux jeux pour les OS modernes avec le matériel moderne comme la prise en charge des écrans large ou de Vulkan, OpenGL 4, DirectX11 et les binaires des jeux (Unreal Gold & UT99 Goty)

UT99 linux native

  1. Pour cela il faut récupérer les deux ISO de UT99 sur archive.org (CD1 & CD2)
  2. Récupérer le dernier patch linux sur le github du projet OldUnreal, dans mon cas c'est le patch dernier en date au moment de la réaction OldUnreal-UTPatch469e-Linux-amd64.tar.bz2
  3. Décompresser le patch
  4. Monter UT_GOTY_CD1
  5. Copier les dossiers suivant du CD vers le dossier du patch sans effacer les fichiers déjà présents :
    • Maps
    • Music
    • Sounds
    • Textures
  6. Faire de même avec UT_GOTY_CD2
  7. Lancer le jeux <votre repetoire UT>/System64/ut-bin-amd64

Wine

dpkg --add-architecture i386 && apt-get update && apt-get install wine32:i386 winetricks

winetricks orm=backbuffer glsl=disable
